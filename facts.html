Do I need to know how to program already?

No, but if you don't then you need to be somewhat motivated to learn. While programming isn't hard, it certainly isn't easy. You need to be willing to learn and a lot of this club is do-it-yourself. We give you the tools to figure stuff out for yourself.

Are we going to learn how to “hack”?

According to Webster, “hack” has the following definitions:

  • a : to write computer programs for enjoyment
  • b : to gain access to a computer illegally

We will be doing a, not b.

What tools do we use?

Whatever you want. So far the club has used:

  • PHP for Web Development
  • Java for some applications
  • Java for Android
  • Xcode + Objective C for iPhone apps
  • Java+Eclipse+Mercurial for GE Case Study

It's easy to learn if you've used anything else (Netbeans, Visual Studio, …) and we'll be glad to provide direction.

Are we going to learn how to make video games?

Maybe. You choose the projects you do. However, bear in mind that making video games is not an easy task whatsoever. If you have no programming experience yet then it is unlikely you will be capable of making anything more advanced than, say, Minesweeper by the end of the year. Perhaps a 2d-sidescroller like Super Mario Brothers. You will not be able to make the next 3d shoot-em-up game.

What competitions do we participate in?

This year we will likely compete in the GE Computer Science Competition (both the case study and the test-if they have it this year), the Quinnipiac Computer Programming Competition, and anything else that might seem interesting. Information will be available ahead of time for those that want to participate. Depending on how many members are interested in participating in the competitions, spaces will be prioritized for those who have completed or are taking Mr. Barretta's AP Computer Science class. Here are links to some information:

  • http://www.ge.com/gecomputerscience/
  • Quinnipiac

Do I need to bring anything?

If you have a laptop that'd be great for you because IT keeps wiping all our settings and so we have to reinstall everything a lot. And if you use your laptop then you can do stuff from home when you feel motivated. We do have computers though if that's inconvenient
