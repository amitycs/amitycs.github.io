So what is the computer science club? What do we do?

Short story: We program. Basically we teach computers the step-by-step instructions for doing something. A simple example would be the following:

# Instruction
1 Check if joystick left is pressed
2 If yes, then move Pac-Man left 3 pixels
3 If PacMan is in contact with a dot:
  • Remove the dot
  • Give PacMan 500 points
4 If PacMan is in contact with a monster:
  • Restart the game
  • Subtract 1 life
5 Go to #1

However, we give the computer instructions in a programming language, which would look something like the following:

              if (joystick[left] == true)
